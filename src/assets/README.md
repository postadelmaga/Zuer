# Font embeddati

`Hack-*.ttf` — famiglia **Hack** (Regular/Bold/Italic/BoldItalic), lo stesso
monospace di default di egui, embeddato nel binario per la resa nativa del testo
(vedi `src/glyph.zig`).

Hack è distribuito sotto la [Hack Open Font License](https://github.com/source-foundry/Hack/blob/master/LICENSE.md)
(MIT + Bitstream Vera), che ne consente l'inclusione e la ridistribuzione.
Upstream: https://github.com/source-foundry/Hack
